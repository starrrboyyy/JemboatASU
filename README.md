
# Mint Bot (ethers.js)

A simple/advanced NFT mint bot for Node.js using ethers.js. Supports:
- simple mode (minimal code)
- single/multi wallet
- retry with gas bump and optional EIP-1559 fees

## Setup
```bash
npm install
cp .env.example .env
# edit .env to match your RPC, contract, price, keys, etc.
```
Run:
```bash
npm start
```
